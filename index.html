About Me

日本での2年間の接客業務では、主に接客・顧客対応・業務報告などを担当しておりました。 その中で、日常的に LINE を活用して業務連絡や顧客対応を行う機会が多くありましたが、 実際に使用する中で、「もっと使いやすく改善できるのではないか」と感じる場面が多くあり、 そこからWebサービスやフロントエンド開発に興味を持つようになりました。

その後、独学で HTML / CSS / JavaScript の学習を開始し、 自身のトレーニング習慣をきっかけに、 日々の運動タスクを記録・管理するための簡易的なフィットネスアプリを制作しました。

現在のアプリはまだ改善点も多くありますが、 今後は IT現場での実務経験を積みながら、 API や React などの技術も学習し、 より多くのユーザーにとって使いやすい、 多機能なフィットネスサービスへと成長させていきたいと考えております。

Projects : 👉Fitness Task Tracker
